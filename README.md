# TitlePE
Easily send title by using command on server !
Usage : /title <player / all> <title> <fadeIn (20 = 1sec)> <duration (20 = 1sec)> <fadeOut (20 = 1sec)>
---
Only working with Pocketmine (don't support Tesseract)
