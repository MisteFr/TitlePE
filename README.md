# TitlePE
Easily send title by using command on server !<br/>
Usage : /title <player / all> <title> <subtitle> <fadeIn (20 = 1sec)> <duration (20 = 1sec)> <fadeOut (20 = 1sec)><br/>
Only working with PocketMine-MP
